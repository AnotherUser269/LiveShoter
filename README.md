# LiveShoter

Приложение для быстрого создания поясняющих скриншотов и анимаций. Позволяет захватывать произвольную область экрана, рисовать поверх изображений и записывать процесс рисования в видео, автоматизируя рутинные операции обработки.

## Возможности

- **Захват области экрана (Start Capturing)** – выделение прямоугольной области через оверлей и сохранение скриншота в папку `Pictures/RawScreenshots`.
- **Статический редактор** – загрузка изображения, рисование кистью (линии и точки), выбор цвета и размера кисти, undo/clear, сохранение, возможность быстро поделиться результатом.
- **Динамический редактор** – то же рисование, но с записью видео (анимация штрихов). Настраиваемая частота кадров.
- **Настройки** – шаблон имени файла (с поддержкой `{time}`), выбор папки сохранения, выбор FPS для видео.

## Требования

- Android 7.0 (API 24) и выше.  
  Для корректной работы оверлея и foreground-сервиса рекомендуется Android 8.0+ (API 26).
- На устройстве должен быть включён сервис MediaProjection.
- Для оверлея требуется разрешение «Поверх других приложений» (SYSTEM_ALERT_WINDOW).
- На Android 13+ дополнительно запрашивается разрешение на уведомления (POST_NOTIFICATIONS).
